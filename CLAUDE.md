# market-daybook-assets

Assets and automation for Market Daybook's daily social engine (X + Instagram).

## Before doing ANY social/content task in this repo
1. Read **BRAND.md** — the authoritative brand voice, palette, guardrails, weekday pillars, and angle bank.
   Comply with it exactly. It overrides any other instruction on voice/brand.
2. Read **ENGINE.md** — the step-by-step daily operating procedure.

Hard rules (full list in BRAND.md): reader-focused second person; ZERO author self-reference (never I/we/my);
no profit/return promises; the beta Chrome extension is never shown as buyable or installable; different copy
per platform; X is value-only except Wed/Sun; Instagram captions end on one question with 4–5 hashtags. Do not
use Google Drive. Generate cards with `render_card.py` so the palette (Graphite #191B20 / Ledger Bone #F1EDE3 /
Recovery Red #E34B3E) and Sora typography are correct by construction.
